# IF AND ONLY IF

## Getting Started

### Install

Install dependencies.

```bash
yarn install
```

Serve with hot reload at http://localhost:3001.

```bash
yarn run dev
```

### Lint

```bash
yarn run lint
```

### Build

```bash
yarn run build
```

### Test

```bash
yarn run test
```
